# Zirelia — Logo Generation Prompts

Use the following prompts to generate logo concepts via AI image tools (Midjourney, DALL-E, Ideogram, Recraft, etc.).

---

## Primary Logo Prompt (Wordmark)

```
Minimal wordmark logo for a tech brand called "ZIRELIA". The letter Z is slightly stylized to suggest movement and digital connection. Clean sans-serif font (Montserrat style). Deep violet (#5A3FD4) on a pure white background. Flat design, no gradients, no shadows. Professional, premium, modern. Suitable for tech SaaS brand. Vector style.
```

---

## Icon-Only Prompt (Abstract Symbol)

```
Abstract minimal logo icon for a brand called Zirelia. Concept: autonomous digital identity. Flat vector, no gradients. Clean, modern, scalable to small sizes. White background.
```

---

## After Generation: QC Checklist

- [ ] Readable at 32x32px (favicon size)
- [ ] Works in both light and dark backgrounds
- [ ] Color matches brand palette (`#5A3FD4` / `#20C997`)
- [ ] No gradients in primary lockup
- [ ] SVG format available for scalability
- [ ] `.png` variants: 512x512, 1024x1024, 1500x500 (Twitter banner)

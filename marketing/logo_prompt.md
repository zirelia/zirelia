## Primary Logo Prompt (Wordmark — Dark Background)

```
Minimal wordmark logo for a tech brand called "ZIRELIA". Clean geometric sans-serif font similar to Space Grotesk or Manrope. The text uses a vivid gradient from violet-purple (#7C3AED / #A78BFA) to hot pink (#EC4899). Dark near-black background (#0D0D0F). Subtle neon glow around the letters in purple. Flat vector style. No shadows. Premium AI brand, dark-first aesthetic. Looks excellent on screens and social media.
```

## After Generation: QC Checklist

- [ ] Readable at 32x32px (favicon size)
- [ ] Works exclusively on dark backgrounds (`#0D0D0F`)
- [ ] Gradient matches brand palette (`#A78BFA → #EC4899`)
- [ ] Neon glow is subtle, not overpowering
- [ ] SVG format available for scalability
- [ ] `.png` variants: 512x512, 1024x1024, 1500x500 (Twitter banner)
- [ ] No light mode version needed (dark-only brand)
